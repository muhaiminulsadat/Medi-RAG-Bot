# Medi-RAG-Bot
Medical RAG Bot is an AI-powered chatbot that answers questions using medical PDF documents as its knowledge base. Simply ask a medical question, and the bot retrieves relevant answers from the uploaded PDFs.

*Features*
- Answers queries from a collection of medical PDFs
- Uses AI for accurate and context-aware responses
- Easy to use interface (Streamlit app)

*Tech Stack*
- Python, Streamlit
- LangChain + Groq API
- Pinecone for vector storage
